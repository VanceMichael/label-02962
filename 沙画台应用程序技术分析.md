#### 粒子生成过程

**生成触发点：绘制画笔绘制时生成粒子：

```python
# canvas.py:497-537
def _spawn_particles(self, pt: QPoint, count: int, strong: bool = False, spray: bool = False):
    # 数量上限保护
    remaining = self.max_particles - len(self.particles)
    count = min(count, remaining)
    
    for _ in range(count):
        angle = random.uniform(0, 2 * math.pi)
        speed = random.uniform(2, 6) if strong else random.uniform(1, 3)
        self.particles.append(SandParticle(
            x=px + random.uniform(-5, 5),
            y=py + random.uniform(-5, 5),
            vx=speed * math.cos(angle),
            vy=speed * math.sin(angle) - random.uniform(1, 3),
            size=random.uniform(1, 3),
            color=QColor(...),
            life=random.randint(30, 60)
        ))
```

**生成策略：

- **随机角度**：0 到 2π 均匀分布
- **速度**：根据画笔类型不同速度范围
- **初始向上偏移**：`vy -= random.uniform(1, 3) 模拟向上抛洒
- **颜色噪声**：沙子颜色基础上 ±10 噪声

#### 粒子运动物理

**物理更新算法**（单次遍历优化）：

```python
# canvas.py:539-579
def _update_particles(self):
    alive = []
    for p in self.particles:
        # 重力 + 摩擦
        p.vy = (p.vy + gravity) * friction
        p.vx *= friction
        # 移动
        p.x += p.vx
        p.y += p.vy
        p.life -= 1

        # 出界或生命结束 → 丢弃
        if p.x < 0 or p.x > w or p.life <= 0:
            continue

        # 底部碰撞 - 沙子堆积
        if p.y >= h - p.size:
            p.y = h - p.size
            p.vy *= -0.3  # 反弹
            p.vx *= 0.8   # 水平阻尼
            if abs(p.vy) < 0.5:
                # 沙子落地，绘制到画布
                canvas.setPixelColor(px, int(p.y), p.color)
                continue  # 落地粒子不保留

        alive.append(p)
    self.particles = alive
```

**物理规则**：

1. **重力加速度**：`vy += gravity
2. **摩擦阻尼**：速度 × friction
3. **边界碰撞**：左右出界删除
4. **底部反弹**：y 方向反弹系数 -0.3，水平阻尼 0.8
5. **静止判定**：`|vy| < 0.5 时落地融合到画布

#### 粒子消亡条件

粒子消亡条件：

1. **生命周期结束**：`life <= 0
2. **水平出界**：`x < 0 or x > w
3. **落地静止**：速度足够小时绘制到画布后删除

### 3.2 性能瓶颈分析与优化建议

#### 当前瓶颈 1：粒子数量瓶颈

**问题**：当前 `max_particles = 500

**瓶颈分析**：

- 每个粒子每帧都要：物理计算 + 绘制
- 500 粒子 × 30fps = 15,000 次/秒
- 每个粒子 7 个浮点运算 + 绘制调用

**如果增加到 2000+ 粒子可能遇到：

1. **CPU 瓶颈**：物理计算耗时
2. **GPU 瓶颈**：绘制调用过多
3. **内存带宽**：粒子数据访问

**优化建议**：

**建议 1：空间分区**

使用 NumPy 向量化计算：

```python
    if not isinstance(intensity, (int, float)):

        raise ValidationError(f"水波纹强度必须是数字")
    
    intensity = float(intensity)
    if intensity < WATER_WAVE_MIN:
        logger.warning(f"水波纹强度 {intensity} 小于最小值，已修正")
        return WATER_WAVE_MIN
    if intensity > WATER_WAVE_MAX:
        logger.warning(f"水波纹强度 {intensity} 大于最大值，已修正")
        return WATER_WAVE_MAX
    
    return intensity
```
