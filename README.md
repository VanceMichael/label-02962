# 沙画台 - Sand Art Table

## How to Run

```bash
# 安装依赖
pip3 install -r backend/requirements.txt

# 运行程序
python3 backend/main.py
```

注意：本项目不使用 Docker 启动，因为这是一个 PyQt6 桌面 GUI 应用程序，需要直接访问本地显示设备和图形界面。Docker 容器默认没有图形界面支持。

## 项目介绍

一个使用 PyQt6 开发的沙画台模拟应用程序，模拟真实沙画创作体验。支持多种画笔工具、粒子系统、物理效果等特性。

## 项目结构

```
.
├── README.md                    # 项目说明文档
├── .gitignore                   # Git忽略文件
└── backend/
    ├── main.py                  # 程序入口
    ├── requirements.txt         # Python 依赖
    ├── models/                  # 数据模型层
    │   ├── __init__.py
    │   ├── brush.py             # 画笔类型枚举
    │   └── particle.py          # 沙粒粒子数据类
    ├── widgets/                 # UI组件层
    │   ├── __init__.py
    │   ├── canvas.py            # 沙画画布组件
    │   └── tool_button.py       # 工具按钮组件
    ├── windows/                 # 窗口层
    │   ├── __init__.py

    └── utils/                   # 工具模块
        ├── __init__.py
```

## 功能特点

### 画笔工具（8种）

| 工具 | 图标 | 说明 |
|------|------|------|
| 手指 | 👆 | 细致绘制，适合描绘细节 |
| 手掌 | 🖐 | 大面积涂抹，椭圆形笔触 |
| 拳头 | ✊ | 强力清除，产生更多粒子飞溅 |
| 耙子 | 🪥 | 5齿并排，绘制平行线条 |
| 梳子 | 〰 | 9齿细密，绘制细密纹理 |
| 喷洒 | 💨 | 随机散点，模拟喷沙效果 |
| 印章 | ⭐ | 图案印章，支持4种图案 |
| 添沙 | ✨ | 添加沙子，恢复沙面 |

### 印章图案（4种）

- ⚪ 圆形
- ⭐ 星形
- ❤ 心形
- 🍃 叶形

### 粒子系统

- 沙粒飞溅效果：绘制时产生沙粒粒子
- 重力模拟：粒子受重力影响下落
- 摩擦效果：粒子速度逐渐衰减
- 堆积效果：粒子落地后绘制到画布

### 其他功能

- 可调节画笔大小（1-100）
- 自定义沙子颜色（含4种预设）
- 自定义画笔颜色（含4种预设）
- 撤销/重做功能（最多30步历史）
- 清空画布
- 重铺沙子
- 导出图像文件（PNG、JPG）

## 技术栈

- Python 3.8+
- PyQt6
- logging（标准库日志模块）

## 工程特性

### 模块化架构

项目采用分层架构设计：

- **models/** - 数据模型层：定义画笔类型、粒子数据结构
- **widgets/** - UI组件层：画布组件、工具按钮组件
- **windows/** - 窗口层：主窗口及界面布局
- **utils/** - 工具层：日志系统、参数校验

### 日志系统

项目使用 Python 标准库 `logging` 模块进行日志记录：

- 日志器名称：`sand_art`（主）、`sand_art.canvas`、`sand_art.window`
- 默认级别：INFO
- 输出格式：`时间 - 模块名 - 级别 - 消息`

日志记录内容：
- 应用启动/退出
- 画布初始化和操作
- 工具切换和参数调整
- 图像保存
- 异常和错误信息

### 参数校验

项目实现了完整的参数校验机制，防止非法输入导致程序异常：

| 参数 | 有效范围 | 处理方式 |
|------|----------|----------|
| 画笔大小 | 1-100 | 自动修正到有效范围 |
| 印章图案索引 | 0-3 | 自动修正到有效范围 |
| 重力值 | 0.0-2.0 | 自动修正到有效范围 |
| 摩擦系数 | 0.0-1.0 | 自动修正到有效范围 |
| 粒子数量 | 0-100 | 自动修正到有效范围 |
| 颜色值 | 0-255 | 自动修正到有效范围 |
| 颜色字符串 | #RGB/#RRGGBB | 抛出 ValidationError |
| 文件路径 | 非空字符串 | 抛出 ValidationError |

## 使用说明

1. 运行程序后，画布默认铺满沙子
2. 左侧面板选择画笔工具（默认为手指）
3. 在画布上拖动鼠标绘制，沙子会被推开显示底部灯光
4. 使用滑块调节画笔大小
5. 右侧面板可自定义沙子颜色和画笔颜色
6. 使用"添沙"工具可以恢复沙面
7. 点击"保存图片"导出作品
